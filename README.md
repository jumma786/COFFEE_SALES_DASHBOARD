# ☕ Coffee Shop Sales Dashboard

A professional sales analytics dashboard project developed using Microsoft Excel to analyze coffee shop transactional data and generate business insights through KPIs, charts, and interactive dashboard reporting.

---

# 📌 Project Overview

This project focuses on transforming raw coffee shop sales data into meaningful business insights using Excel-based business intelligence techniques.

The dashboard helps analyze:
- Sales performance
- Customer purchasing behavior
- Product performance
- Store performance
- Peak operational hours
- Revenue trends

This project demonstrates practical data analytics and dashboard reporting skills commonly used in real-world business environments.

---

# 🎯 Project Objectives

- Analyze overall business sales performance
- Identify top-selling products
- Compare store-level performance
- Detect peak sales hours
- Build KPI-driven dashboards
- Generate actionable business insights

---

# 📂 Dataset Information

## Dataset Type
Transactional Sales Dataset

## Main Columns

| Column Name | Description |
|---|---|
| transaction_id | Unique transaction identifier |
| transaction_date | Date of transaction |
| transaction_time | Time of transaction |
| transaction_qty | Quantity sold |
| store_location | Store branch |
| product_category | Product category |
| product_type | Product type |
| product_detail | Product details |
| unit_price | Product unit price |
| Revenue | Total revenue |

# 📊 Key Performance Indicators (KPIs)

The dashboard includes the following KPIs:

## 💰 Total Revenue
Measures total sales generated.

## 🧾 Total Orders
Measures total customer transactions.

## 📈 Average Order Value (AOV)
Average amount spent per order.

## 🏆 Best Selling Product
Highest-performing product by revenue.

## ⏰ Peak Sales Hour
Busiest operational hour.

## 📍 Best Performing Store
Top revenue-generating branch.
